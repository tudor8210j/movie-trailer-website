Programmer: Jon Tudor
Purpose: To display a web page view of my favorite movies with trailers.
Version: 1.0
Other Contributors/Credits: Udacity Full Stack Developer Nanodegree, Google Image Search and YouTube
QuickStart:

In order to install this program unzip the Movie Trailers Website folder provided by Udacity.
Once unzipped, you can run the movie_trailer_website_entertainment_center.py file to open the web page.
Do this by either opening it via the Python Shell and running it, running it directly from the Python shell or by double clicking on the file.
